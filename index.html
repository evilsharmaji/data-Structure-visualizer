Array

Contiguous memory structure enabling constant-time indexed access.

  • Access: O(1)
  • Insertion: O(n)
  • Use: Static collections

Stack

LIFO structure used in recursion and undo operations.

  • Push/Pop: O(1)
  • Use: Expression evaluation

Queue

FIFO structure for scheduling and buffering.

  • Enqueue/Dequeue: O(1)
  • Use: Task scheduling

Linked List

Nodes connected via pointers allowing dynamic memory usage.

  • Insertion: O(1)
  • Search: O(n)

Hash Table

Key-value storage using hashing for near constant-time operations.

  • Average: O(1)
  • Use: Databases, caching

Binary Tree

Hierarchical structure where each node has at most two children.

  • Traversal operations
  • Use: Searching, sorting
